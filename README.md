# CreditWise Loan Prediction System

## 📌 Project Overview

**CreditWise Loan Prediction System** is a Machine Learning based web application that predicts whether a loan application is likely to be **approved or rejected** based on applicant details.

The system uses historical loan data and a classification model to assist financial institutions in making faster and more data-driven lending decisions.

The application is deployed as an interactive web app using Streamlit.

---

## 🚀 Live Demo

You can try the live application here:

https://creditwiseloanpredictionsystem-siri.streamlit.app

---

## 🧠 Machine Learning Model

The prediction model is built using:

**Logistic Regression**

The model analyzes applicant information and predicts the loan approval status.

---

## 📊 Input Features

The model uses the following features:

* Gender
* Married Status
* Dependents
* Education
* Self Employed
* Applicant Income
* Co-Applicant Income
* Loan Amount
* Loan Amount Term
* Credit History
* Property Area

## ⚙️ Installation and Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Sirisha-Nukala/CreditWise_Loan_Prediction_System.git
```

### 2️⃣ Navigate to the project folder

```
cd CreditWise_Loan_Prediction_System
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
streamlit run app.py
```
